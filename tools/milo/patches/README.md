# MILo Training Patches

Reference files for MILo training customizations. These are NOT copied directly
into the Docker image — instead, build-time patchers modify MILo's original files.

## Files

- `train.py` — Reference training script (base for patcher anchors)
- `scene_init.py` — Clean Scene class (no mask-related code; masks loaded in train.py)

## Build-Time Patchers

Applied in order by `Dockerfile.onyx-milo`:

1. `patch_train_difix.py` — Adds Difix3D+ fix cycles (`--difix3d` flag)
2. `patch_train_masks.py` — Adds mask-aware training (`--masks` flag):
   - Loads mask PNGs onto camera objects after Scene construction
   - SSIM zero-error infill (full quality on unmasked pixels)
   - Densification gradient suppression in masked regions
   - Delayed densification start (500 → 3000 iters)

## Mask Convention

- White (255) = include pixel in training loss
- Black (0) = exclude pixel (transient object)
- Masks generated by `onyx-segformer` in MASKING pipeline step
